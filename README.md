### 📂 这里面都有啥？

这个仓库是个大杂烩，包含了我大学期间折腾过的各种东西：

* **数据结构课程设计**：各种链表、树、图的经典实现。
* **前端小玩具**：什么破玩意都有。
* **数据库课程设计**：SQL、ER 图以及配套的增删改查逻辑。
* **地理信息系统 (GIS) 专项**：
    * **桌面端**：一套利用 **QGIS 开源库** 编写的地理信息系统。
    * **Web端**：一套用 **Cesium** 编写的前端三维地理信息系统（这个挺酷的）。

### 🛠️ 如何开始？

1.  **克隆仓库**：`git clone https://github.com/你的用户名/仓库名.git`
2.  **切换分支**：`git checkout <你要看的分支名>`
3.  **配置环境**：GIS 类的项目记得看子目录里的说明，库没装对是跑不起来的。
